### Database:

All course data is centralized in `CoursePagesConfig.js` making it easy to:
- Update course details
- Add new courses
- Modify testimonials and placements
- Change duration and software lists

### Routing:

Dynamic route accepts `:courseSlug` parameter from URL, matching against the config database for flexible, scalable course page management.

### Implementation:

To view any course page, navigate to:
- `http://localhost:3000/course/graphic-design-uiux`
- `http://localhost:3000/course/advanced-graphic-vfx`
- `http://localhost:3000/course/3d-animation`
- etc.

All pages automatically:
- Load correct course data
- Display proper curriculum
- Show student testimonials
- Display placement info
- Maintain consistent styling
