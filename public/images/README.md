# Image Organization

This folder contains all the images used in the Shankar Multimedia website.

## Folder Structure:

### `/backgrounds/`
- Banner images and background assets
- Homepage hero backgrounds
- Section backgrounds

### `/courses/`
- Course-specific images and icons
- Background images for different courses
- Course category illustrations

### `/team/`
- Team member photographs
- Staff photos
- Instructor images

### `/icons/`
- Social media icons
- UI icons and symbols
- Navigation icons

### `/gallery/`
- Student work samples
- Project galleries
- Portfolio images

### `/placements/`
- Company logos
- Placement success stories
- Achievement certificates

## Image Usage in Components:

All images should be referenced using the public path:
```javascript
// For images in public/images/
<img src="/images/logo.png" alt="Shankar Multimedia Logo" />

// For course images
<img src="/images/courses/3D_bg.png" alt="3D Animation Course" />

// For team photos
<img src="/images/team/anurag_singh.jpg" alt="Anurag Singh" />
```
